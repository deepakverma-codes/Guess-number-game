# 🎯 Guess Number Game

A Java-based Number Guessing Game built using Maven with database integration. The game generates a random number, and the player tries to guess it in the minimum number of attempts. Player scores and game records are stored in the database.

## 🚀 Features

- Random number generation
- Six guessing attempts
- Hint system (Too High / Too Low)
- Stores player data in the database
- Tracks number of attempts
- Built using Maven
- Object-Oriented Programming (OOP)

## 🛠️ Technologies Used

- Java
- Maven
- JDBC
- MySQL

## ⚙️ Setup

1. Clone the repository

```bash
git clone https://github.com/deepakverma-codes/Guess-number-game.git
```

2. Open the project in IntelliJ IDEA or Eclipse.

3. Configure the database connection.

4. Create the required database and tables.

5. Run the project.
